The concept here is a "Scheduled Movie Streaming Management System"

Elements being tested are:

  • Objects,
  • Arrays,
  • Functions,
  • Selection,
  • Full-Scope Variables,
  • Booleans, and
  • API Processing
  • Critical and Procedural Thinking

The first thing that needs to be done is to use your TMDB API Account to pull a list of the top trending movies and preload the movie array (more instructions, as well as Object Structure, are provided in the JS file)

Create an Array of Movies, Showtimes and Prices

In this section we'll:

  • There are Instructions within the JS File that you need to Pay attention to
  • add Unlimited Movies
  • add Seperate Showtimes for each Movie
  • add Price for each Movie at the specified Showtime

The idea here is to add 1 Movie-Showtime-Price Object for each showing of a particular movie and push it into a Movie Array - for example:
(Note - take time as an Integer...no need for AM/PM, can use 24hr clock; Expect decimals for price)

  • Love Actually - 11am - 4.99
  • Love Actually - 2pm - 6.99
  • Love Actually - 5pm - 8.99
  • Love Actually - 8pm - 11.99
  • Notting Hill - 11am - 4.99
  • etc...
